Aloha Editor UI Demo

This UI demo is a basic implementation that you can use as a starting point to integrate Aloha Editor into your own application. Simply view source.

Included is aloha-ui-links.js, which is an additional module that handles UI interactions for editing links. Look at that as a reference for extending and customizing the capabilities of Aloha Editor.

Aloha Editor is a better alternative to ContentEditable

Aloha Editor is a functional content editing library. It has been designed this way to make it a simple and extensible foundation for building—not only word-processing WYSIWYG editors—but future digital native content authoring tools.

  • MS-Word friendly. As you expect it to be.
    Perfect Copy & Paste from MS Word - keep all formatting, lose all the bad code. Click and edit is all you need with Aloha Editor. Always work on the final document. You see what you get - with every keystroke!
  • Easy integration. Created with you in mind.
    Just a few lines of code needed to run it in your application. Additional features? Aloha Editor is a stand alone library with a functional and stateless API that provides you with essential editing capabilities not available in browsers.
  • 60 fps. Ridiculously smooth.
    Our stack executes in 5-6ms for most editing operations. Resulting in a frame rate of 60 FPS. Perfect for creating modern, smooth web applications and boosting its user engagement and user interaction.
  • Lean. Smallest possible filesize.
    Aloha Editor is smaller - with 142KB it's loading quicker and starting faster than other browser based editors.
  • Full control. Aloha Editor is your content editing API.
    No more contentEditable. Other editors are limited by web browsers capabilities. Aloha Editor isn´t. It gives you full control of the editing experience and the content created by it.
  • Your UI is our command. Aloha Editor puts your application first.
    Use the clean and modern user interface of Aloha Editor, customize it or build your own according to your needs. Make it an integral part of your web application and the user experience of your customers.
