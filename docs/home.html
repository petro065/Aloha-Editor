With pure JavaScript & HTML

Aloha Editor is taking a different approach to text editing.

All caret rendering, selection handling, and DOM manipulation is done entirely through JavaScript without going through the browser's native WYSIWYG blackbox. This is not contentEditable!

This means that the algorithms behind all editing operations are directly accessible to developers to reason about, and change if necessary.
